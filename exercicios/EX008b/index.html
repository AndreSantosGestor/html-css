Explicação da tag code: A tag code serve para tornar o texto mais legivel no quesito largura das letras. Todas ficam com o mesmo espaçamento.

A fonte padrão que geralmente é exibida quando a tag code é usada no HTML é a fonte monoespaçada. A fonte monoespaçada é uma fonte em que todos os caracteres possuem a mesma largura, o que facilita a leitura de código e a alinhamento de elementos em uma representação visualmente clara. No entanto, a aparência exata da fonte monoespaçada pode variar dependendo do sistema operacional e das configurações de fonte do navegador.

A tag CODE não indenta as linhas de comando necessitando do envelopamento com a tag chamada PRE.

Exemplo: sem a tag pre

num = int(input('Digite um número')) if num % 2 == 0; print(f'O número {num} é par') Else; print('fim do programa')

Observe que fica só um linha inteira de códigos e não há separação nem indentação

E com a tag pre


num = int(input('Digite um número'))
if num % 2 == 0;
print(f'O número {num} é par')
Else;
print('fim do programa')

Tabulação

Para fazer avançar a tabulação tanto de uma linha de código, quanto do código inteiro, use TAB (avançar e SHIT-TAB (recuar)

Citações simples

Para citações tipo uma fala de outra pessoa, não devemos usualmente digitar as aspas e sim usar a tag q

Exemplo:

Como diria um pai de um amigo: O computador é um burro muito rápido.

Citações completas

O blockquote é uma citação para ser usada quando queremos ela em uma linha separada e com afastamento tabulado carcterístico do próprio comando. isso é muito importante pois não é simplesmente uma tabulação, mas uma tag que tem um sentido de citação. Sendo importante para a semântica

Exemplo sem a tag blockquote:

Segundo Jeff Noble, no seu livro HTML para leigos:

A diferença entre elementos inline e um texto é importante. Os elementos HTML neste capitulo descrevem os blocos de texto.

Exemplo com a tag blockquote:

Segundo Jeff Noble, no seu livro HTML para leigos:

A diferença entre elementos inline e um texto é importante. Os elementos HTML neste capitulo descrevem os blocos de texto.
