A tag (ol) ordena em ordem crescente os itens descritos. Observe que foram 10 ítens, porém, começando do 2 e terminando no 11. Isto é devido ao parâmetro adicionado que é opcional chamado (type) e (start), onde se escolhe o tipo de organização.
As opções são:
(1) que ondena em numeros de forma crescente,
(A) que ordena em letras maiúsculas,
(a) que ordena em letras minusculas,
(I) números romanos maiúsculos,
(i) Numeros romanos minusculos

O parâmetro (start) pergunta à partir de onde se quer iniciar a lista. Observação: só aceita números. Nunca letras.

    Lembrando que:
    Para estilizar (ol), basta usar style= para definir 1, A, a, I, ou i, e start para informar de onde quer começar a lista, 1, 2, 3, 4, 5.....

    E para estilizar (ul), basta usar style= para definir disk, circle e square. (não há start em ul)
