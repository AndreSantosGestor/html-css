Você pode acessar também o seu youtube Canal gratuito no YouTube

O primeiro exemplo é um link para um site externo e o segundo para dentro do mesmo site

Obs: Sempre que o link for para um site externo, deve se usar "target="_blank"rel=external, pois assim o site abrirá em nova Aba e não substituirá o site de origem.

Para criar links, use a tag a

Onde: (href) se refere ao local do link, e onde se lê (repositório público do github), é onde se escreve a descrição e onde deve ser clicado para ir para o outro site externo ou página do próprio site.
